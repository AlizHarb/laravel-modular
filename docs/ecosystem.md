# Ecosystem

Laravel Modular is designed to be the core of a powerful, extensible ecosystem. While the core package provides the architecture, these official add-ons supercharge your development workflow.

## Official Packages

| Package | Description |
| :--- | :--- |
| **[laravel-modular-livewire](https://github.com/alizharb/laravel-modular-livewire)** | **Livewire Integration**<br>Provides automatic component discovery and registration within modules. No manual `Livewire::component()` calls needed. |
| **[laravel-modular-filament](https://github.com/alizharb/laravel-modular-filament)** | **Filament Admin Support**<br>Enables Filament v5 admin panel integration with automatic resource, page, and widget discovery inside modules. |
| **[laravel-modular-js](https://github.com/alizharb/laravel-modular-js)** | **JavaScript Autoloading**<br>Enables JS discovery within modular structures and provides zero-config autoloading for module assets. |
| **[laravel-themer](https://github.com/alizharb/laravel-themer)** | **Theme System**<br>Advanced theme management support. Switch themes at runtime, manage asset pipelines, and more. |
| **[laravel-hooks](https://github.com/alizharb/laravel-hooks)** | **Plugin System**<br>Adds a universal extensibility and hook system (actions/filters) to your Laravel application. |

## Filament Tools

| Package | Description |
| :--- | :--- |
| **[filament-themer-luncher](https://github.com/alizharb/filament-themer-luncher)** | **Theme Manager GUI**<br>A comprehensive Filament v5 interface for previewing, managing, and switching active themes. |
| **[filament-modular-luncher](https://github.com/alizharb/filament-modular-luncher)** | **Module Manager GUI**<br>A powerful Filament v5 manager for listing, toggling, enabling/disabling, and backing up system modules via the admin panel. |

## Installation

You can install any of these packages via Composer:

```bash
composer require alizharb/laravel-modular-livewire
```
